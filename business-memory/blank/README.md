# Business Memory (blank version)

These are the same five business-memory documents as the parent folder, but with the examples
removed — just the sections and instructions, ready for you to type into.

If you'd like to see filled-in examples for reference, look at the versions one folder up in
[`../`](../).

## How to use

1. Fill out each file below. Every section has a short *instruction* telling you what to write and
   why, followed by a blank line marked `→` where your answer goes.
2. At the start of any Claude Co-Work session, drop this folder in (or point Claude at it) so Claude
   has standing context about your business.
3. Keep them current — revisit once a quarter or whenever your offering, pricing, or audience shifts.

## The five pillars

Fill them out in this order — each builds on the one before it:

1. **[company-overview.md](company-overview.md)** — the anchor. Start here.
2. **[ideal-customer-profile.md](ideal-customer-profile.md)** — who you serve and their pain points.
3. **[brand-voice.md](brand-voice.md)** — how you sound and look.
4. **[product-and-positioning.md](product-and-positioning.md)** — what you sell and how you stand apart.
5. **[sales-strategy.md](sales-strategy.md)** — how you acquire and close customers.

If a section doesn't apply to you, write "N/A" so Claude knows it was intentional rather than unfinished.

Setup Guide

AI Answer Readiness Scorer

This tool crawls your website and scores how well AI systems — ChatGPT, Google AI, Claude — can find, read, and cite it. Follow the steps below and you'll have your report in a few minutes.

What you'll get

  • A score out of 100 across 6 categories: crawlability, content clarity, structured data, freshness, and more
  • A plain-English breakdown of what's working and what to fix
  • A saved Markdown report you can share with your team or developer
1
Download & unzip Done ✓

If you're reading this, you already did it. The file ai-answer-scorer.zip is unpacked and ready to go.

2
Open Claude Code

Claude Code is the AI assistant that will handle all the technical setup for you — no terminal experience needed.

If you don't have it yet, download it free from claude.ai/code and sign in with your Anthropic account.

3
Drag INSTALL.md into Claude Code

In this folder, find the file called INSTALL.md. Drag it directly into the Claude Code window — the same way you'd share a file in a chat.

Where to find it It's in the same folder as this page — look for INSTALL.md right next to START_HERE.html.
4
Say the magic words

After dragging the file in, type this message and hit send:

Paste this into Claude Code Please follow the instructions in this file

Claude will take it from there — installing any dependencies, asking for your website URL, running the scan, and delivering your report.

5
Get your report

Claude will scan up to 30 pages of your site (takes 1–3 minutes) and generate a detailed Markdown report with your scores and recommendations.

Pro tip The report includes an AI Scoring section at the bottom. Open it in a new Claude conversation and say "please score the content sections" to unlock the remaining 35 points out of 100.

Questions? The INSTALL.md file has full troubleshooting details — just share it with Claude and ask.
